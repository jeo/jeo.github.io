Library Files

All the the jar files for jeo and its dependencies.

  • jeo-{{site.version}}-lib.zip
Alternatively the library can be obtained using a build tool like Maven or Gradle. See the docs for more details.

Command Line Tools

The jeo command line tools. Packages are available for a number of platforms.

  • Mac
  • Ubuntu
  • CentOS
  • Fedora
  • Windows

Mac users can install with Homebrew.

  1. Tap and install with brew

    {% highlight bash %} $ brew tap jeo/jeo $ brew install jeo {% endhighlight %}

  2. Run it. {% highlight bash %} $ jeo --version {% endhighlight %}

See the docs for additional notes about installation on osx.

Ubuntu packages are available for precise and trusty 64 bit.

  1. Download jeo_{{site.version}}_amd64.deb and install with dpkg

    {% highlight bash %} $ wget {{site.download_url}}/jeo-cli/releases/download/{{site.version}}/jeo_{{site.version}}_amd64.deb $ sudo dpkg -i jeo_{{site.version}}_amd64.deb {% endhighlight %}

  2. Run it. {% highlight bash %} $ jeo --version {% endhighlight %}

See the docs for additional notes about installation on Ubuntu.

RPM packages are available for CentOS 6 64 bit.

  1. Download jeo-{{site.version}}-1.el6.x86_64.rpm and install with rpm:

    {% highlight bash %} $ wget {{site.download_url}}/jeo-cli/releases/download/{{site.version}}/jeo-{{site.version}}-1.el6.x86_64.rpm $ sudo rpm -ivh jeo-{{site.version}}-1.el6.x86_64.rpm {% endhighlight %}

  2. Run it. {% highlight bash %} $ jeo --version {% endhighlight %}

See the docs for additional notes about installation on CentOS.

RPM packages are available for Fedora 21+ 64 bit.

No native packaging is available for Windows at this time. However jeo can be installed manually on Windows via the platform independent archive.

  1. Ensure a Java 8 runtime is installed. Installers can be downloaded from Oracle. Ensure java is on the Windows PATH or set the JAVA_HOME environment variable.

  2. Download and unpack jeo-{{site.version}}-cli.zip

  3. Updated the Windows PATH to include the bin directory from the unpacked archive.

  4. Run it. {% highlight batch %} $ jeo --version {% endhighlight %}
