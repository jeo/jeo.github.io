The jeo command line tools. Packages are available for the platforms listed below.

  • Mac
  • Linux
  • Other

Library Files

All the the library files for jeo and its dependencies.

  • jeo-{{site.version}}-lib.zip

Mac users can install with Homebrew. {% highlight bash %} $ brew install jeo {% endhighlight %}

Ubuntu

Ubuntu users can install with apt. Start by adding the jeo apt repository. The following commands must be run as root. {% highlight bash %} $ wget -q0- http://apt.opengeo.org/gpg.key | apt-key add - $ echo "deb http://apt.opengeo.org/jeo/ubuntu" > /etc/apt/sources.list.d/jeo.list {% endhighlight %} Then install the jeo package. {% highlight bash %} $ apt-get update $ apt-get install jeo {% endhighlight %}

Fedora/CentOS/Redhat

RPM based distro users can install with yum. Start by adding the jeo yum repository for your distro. The following commands must be run as root. Fedora distros: {% highlight bash %} $ wget http://yum.opengeo.org/jeo/fedora/jeo.repo -o /etc/yum.repos.d/jeo.repo {% endhighlight %} CentOS/RedHat distros: {% highlight bash %} $ wget http://yum.opengeo.org/jeo/centos/jeo.repo -o /etc/yum.repos.d/jeo.repo {% endhighlight %} Then install the jeo package. {% highlight bash %} $ yum install jeo {% endhighlight %} Upon successful installation the jeo command should be available. {% highlight bash %} $ jeo --version {% endhighlight %}
